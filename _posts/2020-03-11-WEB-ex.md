### 테이블 생성
테이블 속성 : 번호,이름,국어점수,수학점수,영어점수 (번호는 primary key)  
  
`score 테이블 생성 SQL 구문`
~~~
create table score(
    no number(3) primary key,
    name char(12) not null,
    korean number(3) not null,
    english number(3) not null,
    math number(3) not null 
    );
~~~
