## for문 연습 문제 모음

### 01. 구구단
N을 입력받은 뒤, 구구단 N단을 출력하는 프로그램을 작성하시오.

~~~cpp
import java.util.Scanner;

public class Main{
    public static void main(String args[]){
        Scanner sc = new Scanner(System.in);
        
        int N = sc.nextInt();
        
        for(int i=1; i<10; i++){
            System.out.println(N + " * " + i + " = " + N*i);
        }
        sc.close();
    }
}//Main
~~~
